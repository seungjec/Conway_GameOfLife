# Conway_GameOfLife  

![img](doc/Conway_GameOfLife.png)

[Usage]  
- Mouse Left Button : Add Cell  
- Mouse Right Button : Remove Cell  
- Keyboard Spacebar : Pause  
- Keyboard Tab : Restart with random cell position
- Keyboard ECS : Quit
